# OpenAI-Article-Summarizer

## Description

This is a simple Article Summarizer

## Main Stacks

React JS\
Vite\
Tailwind Css\
Rapid API\

## System Requirements
React
Node 

## How to run this app

Go to the folder.
Run npm `install @reduxjs/toolkit` .
Install the Tailwind Css 
Run in your local with `npm run dev` start.
